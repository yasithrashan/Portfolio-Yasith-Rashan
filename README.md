# Portfolio - Yasith Rashan

Personal portfolio website built with Next.js and TypeScript.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the portfolio.

## Features

- Responsive design
- Smooth animations
- Modern UI
- Fast performance
